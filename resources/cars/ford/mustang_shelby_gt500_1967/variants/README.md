The production catalog contains exactly two playable variants.

- stable variant ID: `ford_mustang_shelby_gt500_1967_4mt`;
- resource: `gt500_428_4mt.tres`;
- player scene: `res://scenes/cars/mustang_shelby_gt500_1967_4mt.tscn`;
- engine label: `428 FE Police Interceptor 7.0 V8, dual 4V`;

- default variant: yes;
- AI eligibility: no, pending a dedicated AI scene and AI-appropriate engine-audio resource.

Both variants use the completed shared detailed-wheel controller and the same sampled 428 FE engine curve.

## Excluded from this model

The one-off 427 FE Super Snake test car is not a third production variant. The 1968 G.T. 500KR 428 Cobra Jet, G.T. 350 engines, regular Mustang engines, modern crate engines and restomod transmissions are also outside this model's historical scope.
