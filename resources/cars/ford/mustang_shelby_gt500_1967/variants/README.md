# 1967 Shelby G.T. 500 production variants

The production catalog boundary contains exactly two planned variants.

## Four-speed manual

- stable variant ID: `ford_mustang_shelby_gt500_1967_4mt`;
- planned resource: `gt500_428_4mt.tres`;
- engine label: `428 FE Police Interceptor 7.0 V8, dual 4V`;
- transmission: Ford four-speed manual, Toploader family;
- drivetrain label: `RWD`;
- proposed default variant: yes.

## Three-speed automatic

- stable variant ID: `ford_mustang_shelby_gt500_1967_3at`;
- planned resource: `gt500_428_3at.tres`;
- engine label: `428 FE Police Interceptor 7.0 V8, dual 4V`;
- transmission: Ford C6 SelectShift Cruise-O-Matic three-speed automatic;
- drivetrain label: `RWD`;
- proposed default variant: no.

## Excluded from this model

The one-off 427 FE Super Snake test car is not a third production variant. It may only be added later as a separately named prototype with independent research and complete runtime resources.

The 1968 G.T. 500KR 428 Cobra Jet, G.T. 350 engines, regular Mustang engines, modern crate engines and restomod transmissions are outside this model's historical scope.

No `.tres` variant should be added before its player scene, AI scene where applicable, valid `CarSpecs` resource and complete presentation metadata are ready together.
